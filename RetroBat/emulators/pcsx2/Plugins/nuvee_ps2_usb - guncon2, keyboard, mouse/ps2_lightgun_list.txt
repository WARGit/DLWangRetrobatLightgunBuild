(USB) Guncon 2 compatible


IMPORTANT README:
- At calibration screen, hold trigger offscreen + press SELECT

  change screen: 480i, 480p, 100 Hz
  ('Progressive Scan' is 480p)

- Progressive scan offers higher precision


- Keyboard D-pad mapping:
W = up
A = Left
S = Down
D = Right

Q = Start
E = Select
F = Button C
